世界最速のウェブサイト

  • English version here

概要

このウェブサイトは、パフォーマンス、アクセシビリティ、ミニマリズムを重視し、世界最速を目指して設計されています。個人のウェブサイトとしての独自の個性を維持しながら、日本のデジタル庁のデザインシステムからインスピレーションを得ています。

パフォーマンスの最適化

コンテンツ配信ネットワーク (CDN)

このウェブサイトは、最適なコンテンツ配信のために Vercel CDN を利用しています。Vercel のエッジ ネットワークにより、静的アセットが地理的に分散した場所から提供されるため、世界中のユーザーの待ち時間が最小限に抑えられ、読み込み時間が短縮されます。

ミニマリストデザイン

このウェブサイトはミニマリストなデザインを採用しており、転送する必要のあるデータの量を削減しています。この方法は読み込み時間を短縮するだけでなく、重要なコンテンツに焦点を合わせることでユーザーエクスペリエンスも向上させます。

静的 HTML

静的 HTML を使用することで、サーバー側の処理とデータベース クエリの必要性がなくなり、最初のバイトまでの時間 (TTFB) と全体的な読み込み時間が大幅に短縮されます。

CSS 最適化

CSS は、望ましいデザインを維持しながら、可能な限り小さくになるように細心の注意を払って作成されています。 ショートカット プロパティや冗長なルールの回避などの手法が採用され、CSS ファイル サイズが最小限に抑えられています。

JavaScriptなし

このウェブサイトは JavaScript なしで機能し、クライアント側のスクリプトに通常伴う解析、コンパイル、実行時間を排除します。この決定により、クライアントのデバイス上の計算負荷が大幅に軽減され、読み込み時間がさらに短縮されます。

アクセシビリティ

色のコントラスト

サイト全体の色のコントラスト比には細心の注意が払われています。背景色とテキスト間の高コントラストにより、読みやすさと WCAG 2.1 ガイドラインへの準拠が確保されます。これには次のものが含まれます。

  • 背景とテキストのコントラスト
  • ボタンの色とテキスト
  • 周囲のコンテンツとリンクカラー

テキスト

このサイトでは、「Fira Code」から始まる等幅フォント スタックを使用しており、明瞭で読みやすいテキストを提供しています。フォント サイズと行の高さは、さまざまなデバイスや画面サイズで最適な読みやすさが確保されるように設定されています。

デザインのインスピレーション

このウェブサイトは、独自の個性を維持しながら、日本のデジタル庁のデザインシステムからインスピレーションを得ています。この影響は、次の点に顕著に表れています。

  • すっきりとしたミニマリストレイアウト
  • テキストと読みやすさに重点を置く
  • 色のコントラストとアクセシビリティへの配慮

技術仕様

HTML

このウェブサイトはセマンティック HTML5 を使用しており、適切な構造と改善されたアクセシビリティを確保しています。各ページは静的に生成されるため、読み込みとレンダリングが高速です。

CSS

スタイル設定には、最適化された単一の CSS ファイルが使用されます。レイアウト用の flexbox や、一貫した配色を維持するためのカスタム プロパティなど、最新の CSS 機能が採用されています。

ホスティング

サーバーでは、グローバル エッジ ネットワークを活用して最適なパフォーマンスを実現しています。この設定により、次のことが可能になります。

  • 自動HTTPS暗号化
  • グローバルコンテンツ配信
  • デプロイメント時の即時キャッシュ無効化

パフォーマンス比較

この Web サイトと dev.to の詳細なパフォーマンス比較は、DebugBear を使用して実施されました。 この分析により、読み込みプロセス、レンダリング時間、全体的なパフォーマンスに関する洞察が得られます。
